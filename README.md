# .NET Developer Journey

## Repository Overview

This repository contains my daily learning, coding exercises, notes, hands-on tasks, mini projects, and final project completed throughout my **8-Week .NET Developer Journey**.

The goal of this journey is to build a strong foundation in **C#**, **Object-Oriented Programming (OOP)**, **SQL Server**, **Entity Framework Core**, **ASP.NET Core MVC**, **ASP.NET Core Web API**, and **Authentication & Authorization** through structured learning and practical implementation.

Each day's work is organized into a separate folder containing the source code, notes, and a dedicated `README.md` documenting the concepts learned and the work completed.

This repository will be updated regularly throughout the journey to track my progress, improve my problem-solving skills, strengthen my understanding of .NET development, and showcase my growth toward becoming a **Junior .NET Developer**.
